Penjelasan Button :
1. SIMPAN
Button simpan dibuat untuk memasukkan data kedalam file txt bernama datanapi.txt
Banyak data yang bisa dimasukkan sebanyak 100. 
Cara memasukkannya cukup ketik data pada text box. Semua field harus di isi karena jika ada yang kosong
akan keluar pesan error.
Setelah semua data diketik, klik simpan untuk menyimpan data pada datanapi.txt kemudian data tersebut akan di tampilkan
pada form list.

2. RESET
Button reset digunakan untuk membersihkan semua data yang ada pada field box. Digunakan ketika selesai memasukkan 
data, maka field box harus dibersihkan untuk memasukkan data yang lainnya. Reset juga digunakan untuk mengisi field box
pada saat ingin mengganti data dengan yang baru. Jadi field box harus dibersihkan terlebih dahulu.

3. HAPUS
Button hapus digunakan untuk menghapus data pada form list maupun database. Cara menggunakannya, klik data yang ingin 
dihapus pada form list. Kemudian tekan button hapus maka data akan terhapus.

4. GANTI
Button ganti digunakan untuk mengganti data yang sudah ada pada form list. Cara kerjanya, klik data yang akan diganti
pada form list, kemudian isi semua data baru pada field masing-masing. Kemudian klik ganti maka data yang ada di form 
list akan terganti dengan data yang sudah dimasukkan kedalam field box.
